# Overview
This project should be carried out according to the following guidelines:
* Back-end only
* Golang
* Json data
* MVC structure
* MySQL Database
* jwt auth
* GORM
* with a Readme and a Documentation

As defined by the teacher, the raw features intended are (at least):
* ability to connect
* create an account
* create an article
* edit an article
* comment an article
